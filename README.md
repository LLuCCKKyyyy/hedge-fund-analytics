# Hedge Fund Analytics Platform 
 
## 项目描述 
对冲基金投资分析和财务指标追踪Web应用 
 
## 技术栈 
- 前端：React, TypeScript, Ant Design 
- 后端：Python, FastAPI 
- 数据分析：ECharts 
 
## 本地开发 
 
### 前端 
\`\`\`bash 
cd frontend 
npm install 
npm start 
\`\`\` 
 
### 后端 
\`\`\`bash 
cd backend 
python -m venv venv 
source venv/bin/activate  # Windows: venv\Scripts\activate 
pip install -r requirements.txt 
uvicorn app.main:app --reload 
\`\`\`
